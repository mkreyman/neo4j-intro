# neo4j-intro slides

1. Install showoff

        gem install showoff

2. Pull this repo
 
        git pull https://github.com/mkreyman/neo4j-intro.git

3. Run the presentation

        cd neo4j-intro
        showoff serve


P.S. If `showoff` gives you headaches, just download the [pdf](https://github.com/mkreyman/neo4j-intro/blob/master/neo4j-intro.pdf). :)
